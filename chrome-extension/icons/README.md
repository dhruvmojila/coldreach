# Extension Icons

SVG placeholder icons are generated by `scripts/generate-icons.mjs`.

**For the Chrome Web Store**, you need PNG icons (not SVG):

1. Convert SVG → PNG using one of:
   - Inkscape: `inkscape --export-type=png icon128.svg`
   - ImageMagick: `convert icon128.svg icon128.png`
   - Online: https://svgtopng.com

2. Replace icon16/48/128.svg with .png files
3. Update `manifest.json` icon paths from `.svg` to `.png`

For development (unpacked extension), SVGs work fine.
